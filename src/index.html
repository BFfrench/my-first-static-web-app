The Default User

Desigining Simplicity for Science and Productivity

The ethos of 'The Default User' is to develop applications focused on:

  • specialist scientific and productivity applictions.
  • optimal user experience.
  • web-based multi-device / platform support.

The aim is to develop and deploy highly effective and easy to use applications free of charge. For users who would like to 'pay it forward', there will be options to upgrade application aesthetics for small financial contributions.

A list of the currently deployed and under construction applications can be viewed at here: Applications for the default user.
